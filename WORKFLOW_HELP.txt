INGREDIENTS SHEET BUILDER - QUICK HELP
======================================
Build: 2026-06-25 #1  (location_height_percent + location_fit_bars)

WHAT THIS NODE MAKES
--------------------
A single reference sheet for the LTX-2.3 IC-LoRA Ingredients model:
  - A top ROW of character/prop panels (each shown whole, no crop, no squish)
  - A full-width LOCATION band at the bottom (or top)
  - Black background, even gaps - matches Lightricks' own example sheets

OUTPUTS
-------
  sheet_image             -> feed this to the Ingredients model
  reference_sheet_prompt  -> the prose prompt for the sheet
  labeled_preview         -> same sheet with labels, for your own checking

THE SETTINGS YOU'LL TOUCH MOST
------------------------------
  output_width / output_height
      Leave at 1456 x 825 (the on-spec sheet size). The output VIDEO is
      smaller (~768x448); the pipeline downscales the bigger sheet.

  location_height_percent   (default 40)
      How much of the sheet height the LOCATION band gets.
      Character row fills the rest. Bump UP if the location looks squished.

  location_fit_bars         (default OFF)
      OFF = location fills the band, crops its top/bottom if shapes differ.
      ON  = whole location shown, black bars fill the leftover space.
      Turn ON if a wide location (like a beach) is getting cut off.

  panel_gap                 (default 12)
      Black gap between panels and between the row and the location band.
      Keep it > 0 so the model doesn't blend things or carry a seam line.

COMMON FIX: BEACH / WIDE LOCATION GETTING CUT OFF
-------------------------------------------------
  1. Set location_height_percent to about 50
  2. Turn location_fit_bars ON
  -> the whole location shows, with black bars instead of a crop

DROPPING A PANEL
----------------
  Mute the Load Image node feeding that panel. The layout adjusts.

NAMING A CHARACTER CONSISTENTLY
-------------------------------
  Use the id_1..id_6 fields (and location_id). The name is threaded through
  the description so the same character is referred to the same way in every
  panel.

LTX-2.3 FRAME COUNT REMINDER (24 fps)
-------------------------------------
  Frame count must be 8n+1 (e.g. 25, 49, 73, 97, 121, ... 241).
  Quick map at 24 fps:  5s=121   8s=193   10s=241
  Model is happiest UNDER 257 frames (~10.7s). Past ~12-15s, subjects can
  drift toward center; render in shorter segments and extend instead.

CONFIRMING YOU HAVE THE RIGHT VERSION
-------------------------------------
  Watch the ComfyUI startup console for:
      [Ingredients Sheet Builder] loaded build: 2026-06-25 #1 ...
  Or open node.py and read the NODE_BUILD line near the top.
